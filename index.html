MsgBox.JS

A customizable message box function in JavaScript, inspired by the simplicity of Visual Basicâ€™s MsgBox. Enhance user interaction with personalized prompts, multiple button options, and flexible modes.

Usages

MsgBox(prompt, buttons, title, modes)

Parameters

prompt

The message to be displayed in the message box.

buttons

Defines the set of buttons to be displayed in the message box.
Options include: btnOkOnly, btnCancelOnly, btnOkCancel, btnYesNo, btnYesNoCancel, btnRetryCancel, btnIgnore.

title

The text to be displayed in the title bar of the message box.

modes

Defines the icon to be displayed in the message box. Options include: 'info', 'excl', 'crit'. If no mode is specified, a default icon is used.
