MsgBox.JS

Parameters

  • prompt: A string that will be displayed as the message in the message box.

  • buttons: A string that determines the buttons to be displayed in the message box.
    The options are: btnOkOnly, btnCancelOnly, btnOkCancel, btnYesNo, btnYesNoCancel, btnRetryCancel, and btnIgnore.

  • title: A string that will be displayed as the title of the message box.

  • modes: A string that determines the icon to be displayed in the message box.
    The options are: info, excl, crit, and default.

Description

This function creates a custom message box with the given prompt, buttons, title, and icon. The message box is appended to the body of the parent document.


Detailed Behavior

  1. A container for the message box is created and given the class name MsgBoxContainer.

  2. A title box is created within the container, with an icon and a title text. The icon's source URL and alternative text are determined by the modes parameter.

  3. A prompt text is created within the container, displaying the prompt string.

  4. A button box is created within the container. The buttons within this box are determined by the buttons parameter. Each button has an onclick event that removes the message box from the parent document's body and logs the value of the button.
